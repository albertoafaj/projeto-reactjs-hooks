# Projeto do gerador de frases do Naruto

Repositório do projeto do gerador de frases do Naruto, desenvolvido como parte do curso de Introdução aos React Hooks da Digital Innovation, com intuito de reforçar o conhecimento em React e utilização de testes unitários.

Projeto desenvolvido por https://github.com/celso-henrique e publicado em https://github.com/celso-henrique/naruto-quotes-client

## Demo:

![Aplicativo de frases do naturo](projeto_frases_naruto.PNG)

## Instação:
Clonar o repository e instalar as dependencias:
```
$ npm install
```
## Rodando a aplicação:
```
$ npm start
```
## Baixar repositório e instalar a API:
